# AGEN-OHADA

Application de Gestion d'Étude Notariale conforme aux normes OHADA.

## Installation

1.  Créer un environnement virtuel : `python -m venv venv`
2.  Activer l'environnement : `.\venv\Scripts\activate`
3.  Installer les dépendances : `pip install -r requirements.txt`

## Lancement

`rio run` (à confirmer selon la structure du projet)
